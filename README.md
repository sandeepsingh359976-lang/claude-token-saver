# 🚀 Claude Token Saver (Caveman Mode)

Claude is powerful — but it can quickly burn through tokens due to verbose responses.
**Caveman Mode fixes that.**
✅ Short, direct outputs
✅ Less fluff, more signal
✅ Up to ~70% lower token usage
## 🧠 What is “Caveman Mode”?

A lightweight custom skill that forces Claude to respond in a simplified, minimal style.

Think: > Fewer words → Same meaning → Lower cost
## ⚙️ Setup (2 Minutes)

### Step 1 : Go to Claude

### Step 2 : Navigate to: Customize → Skills
### Step 3 : Click: +
### Step 4 : Upload: caveman-mode.zip
### Step 5 : Enable the skill and start using Claude 🎉

## 📊 Before vs After

| Mode          | Output Style      | Token Usage |
| ------------- | ----------------- | ----------- |
| Normal Claude | Verbose, detailed | High        |
| Caveman Mode  | Short, concise    | Low         |
